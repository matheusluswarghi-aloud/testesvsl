# testesvsl

Testes de VSL (Video Sales Letter) — landing pages com vturb smartplayer + Meta Pixel para experimentos de conversão.

## Estrutura

- `index.html` — página VSL atual
- `index.html.backup-v3` — backup de versão anterior
- `codigo_embed_video.txt` — snippet de embed do vturb
- `codigo_otimizador_video.txt` — snippet de otimização

## Deploy

Hospedado em `testesvsl.inglessemtrava.com`. Deploy via push para `main`.
